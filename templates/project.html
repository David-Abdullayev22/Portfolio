FYO - The announcement site

  • Genre: Website
  • Release date: 21/02/25
  • Link: private

My first website written in Django. I learned Django and wrote my first website on it. This is a website where you can post a sale announcement. The site has a lot of functionality, some of the useful features of the site:

  • 1) Registration and authorization. There are user mechanics on this site
  • 2) Password recovery via E-mail and mail server
  • 3) Create and edit. You can specify different data for the ad. You can also upload multiple categories and select a category with a tag
  • 4) Easy sorting on the website by categories and tags
  • 5)The ability to edit an ad if you are its author
  • 6)Convenient information about the product
  • 7) Implemented a system of likes and adding posts to the Favorite tab
  • 8) Feedback system
  • 9)User-friendly user profile
  • 10) Light and dark website theme
  • 11) I wrote all the styles and, in principle, all the code myself
